# sshebra

ssh-server with custom commands inside.

Inspired by https://github.com/shazow/ssh-chat and https://github.com/quackduck/devzat.

## Description

```text
go run .
```

```
ssh doesntmatter@localhost -p 4242 -i ./mykey
...
```
